# Todo CLI

Консольное приложение для управления списком задач.

## Возможности

- Добавление задач
- Просмотр списка задач
- Отметка задач как выполненных
- Удаление задач

## Запуск

```bash
python -m app.main

## CLI Interface

The application provides a simple command-line interface for task management.
